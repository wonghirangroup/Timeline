# Product

## Register

product

## Users

HR Admins และ Managers ของบริษัท SME ไทย ใช้งานในบริบทสำนักงาน — จัดการกะ, อนุมัติวันลา, ดูรายงานเช็คชื่อพนักงาน ผ่าน Web App (Admin/Manager) และพนักงานเช็คอินผ่าน Line LIFF

งานหลักที่ต้องทำในแต่ละวัน: ดูว่าใครมาสาย, อนุมัติหรือปฏิเสธคำขอลา, ตรวจสอบรายงานประจำวัน/เดือน

## Product Purpose

TimeLine HR คือระบบ SaaS แบบ multi-tenant สำหรับ Vendor ที่ขายให้บริษัทลูกค้าหลายบริษัท ระบบจัดการ: Shift-based attendance (ผ่าน Line LIFF), Branch management, Leave & Holiday management

สำเร็จเมื่อ: HR Admin ใช้งานได้ทันทีโดยไม่ต้องฝึก, ดูข้อมูลได้เร็ว, ลดภาระงานซ้ำซากด้านเอกสาร

## Brand Personality

เรียบ มั่นใจ น่าเชื่อถือ — Calm & Minimal สำหรับ context ไทย SME ที่ต้องการ tool ที่ใช้งานได้จริง ไม่หวือหวาเกินไป ไม่ formal เกินไป รู้สึกถึงความเชี่ยวชาญโดยไม่ต้องโชว์

## Anti-references

SAP / Oracle HR — หนัก ซับซ้อน เก่า ใช้ expert เท่านั้น ไม่เป็นมิตร UI ที่ information overload, form-heavy โดยไม่มี hierarchy ชัด

## Design Principles

1. **Clarity over cleverness** — ข้อมูลที่ HR ต้องการต้องเจอได้ใน 1-2 คลิก ไม่ซ่อนอยู่ใต้ dropdown ซ้อนกัน
2. **Thai context first** — typography, spacing และ information density ออกแบบสำหรับ content ภาษาไทยที่มี line-height และ character width ต่างจากภาษาอังกฤษ
3. **Calm authority** — สีและ hierarchy ให้ความรู้สึกควบคุมได้ มั่นใจได้ ไม่ตื่นตาตื่นใจ
4. **Workflow first** — ทุก component ออกแบบจาก use case จริง (เช็คชื่อ, อนุมัติลา) ไม่ใช่ showcase
5. **Mobile-aware** — Manager บางคนดูบน phone ต้องใช้งานได้ครบบน mobile เช่นกัน

## Accessibility & Inclusion

ไม่มีข้อกำหนดพิเศษ ยึด WCAG 2.1 AA เป็นเกณฑ์พื้นฐาน (contrast ratio 4.5:1+ สำหรับเนื้อหาหลัก)
